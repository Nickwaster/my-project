Les bases de Git

git init

Cette commande permet d'initialiser git sur le projet voulu

git status

Nous donne le status des fichiers:

  • untracked (non suivi): le fichier n'est pas suivi par git
  • added (ajouté): le fichier vient d'être ajouté à la zone de suivi
  • modifié (not staged): le fichier est modifié, mais ces modifications ne feront pas partie de la prochaine sauvegarde
  • modifié (staged): le fichier est modifié, ET ces modifications feront partie de la prochaine sauvegarde

git add monFichier

Cette commande ajoutte monFichier à la zone de suivi

git commit -m "commentaire à ajouter"

Cette commande permet de faire une sauvegarde de notre travail à cet instant précis. Ne sera sauvegardé que si qui est présent dans la zone de suivie

git remote add origin http://monurl.com

Cette commande permet d'ajouter notre url (http://monurl.com) à git. Cette url aura pour surnom "origin".

git branch -M main

Cette commande permet de renommer notre branche courante en "main"

git push -u origin main

Cette commande permet d'envoyer le contenu de notre repertoire en ligne sur l'url conservé dans "origin". En a précisé qu'on a envoyer (push) notre branche "main".
